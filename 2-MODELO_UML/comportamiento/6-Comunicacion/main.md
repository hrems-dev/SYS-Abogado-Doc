# Diagramas de Comunicación - Sistema SYS-Abogado-Doc

Esta carpeta contiene los diagramas de comunicación correspondientes a los principales módulos funcionales del sistema **SYS-Abogado-Doc**. Estos diagramas representan el intercambio de mensajes entre los objetos y componentes del sistema durante la ejecución de cada proceso.

## Diagramas de Comunicación

| Código | Módulo | Descripción |
|:------:|---------|-------------|
| **DC01** | Área Cliente | Representa la comunicación entre el cliente y el sistema para consultar el estado de sus casos, gestionar documentos y comunicarse con el abogado. |
| **DC02** | Sistema de Citas | Describe la interacción entre el cliente y el sistema para programar, confirmar y gestionar citas legales. |
| **DC03** | Consulta Legal | Muestra el intercambio de mensajes durante el registro y envío de una consulta jurídica. |
| **DC04** | Gestionar Perfil | Representa la comunicación necesaria para visualizar y actualizar la información del perfil del usuario. |
| **DC05** | Publicaciones | Describe la interacción entre el abogado, el administrador y el sistema para crear, editar, publicar y eliminar contenido. |
| **DC06** | Servicios Legales | Representa la comunicación entre visitantes, clientes y administradores para acceder a los servicios ofrecidos por el sistema. |
