# Diagramas de Comunicación

## Descripción

En esta sección se documentan los **diagramas de comunicación** del sistema **SYS-Abogado-Doc**. Estos diagramas representan la interacción entre los objetos y componentes del sistema mediante el intercambio de mensajes, mostrando cómo colaboran para ejecutar cada proceso funcional.

A diferencia de los diagramas de secuencia, los diagramas de comunicación enfatizan las relaciones entre los participantes y el flujo de mensajes intercambiados durante la ejecución de cada caso de uso.

---
