# Diagramas de Secuencia

## Descripción

En esta sección se documentan los **diagramas de secuencia** del sistema **SYS-Abogado-Doc**. Estos diagramas representan la interacción entre los actores y los componentes del sistema, mostrando el intercambio de mensajes durante la ejecución de cada proceso funcional.

Cada diagrama permite comprender el orden cronológico de las operaciones realizadas por los usuarios y el sistema, facilitando el análisis del comportamiento de los módulos implementados.

---
