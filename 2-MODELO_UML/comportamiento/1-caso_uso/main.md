# Casos de Uso

## Descripción

En esta sección se documentan los **casos de uso** del sistema **SYS-Abogado-Doc**. Estos diagramas representan las funcionalidades principales del sistema desde la perspectiva de los actores que interactúan con él, mostrando los servicios que el sistema ofrece para satisfacer los diferentes requerimientos.

Cada caso de uso permite comprender las responsabilidades de los actores y las funciones disponibles en cada módulo, facilitando el análisis funcional y el diseño del sistema.

---

## Descripción

- Cliente

### Objetivo

Representar las funcionalidades disponibles para el cliente dentro de su área personal.
