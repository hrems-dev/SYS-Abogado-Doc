# Casos de Uso - Sistema SYS-ABOGADO

Esta carpeta contiene los diagramas de casos de uso correspondientes a los principales módulos funcionales del sistema **SYS-ABOGADO**.

## Casos de Uso

| Código | Caso de Uso | Descripción |
|:------:|-------------|-------------|
| **CU01** | Área Cliente | Permite al cliente acceder a su espacio personal para consultar el estado de sus casos, gestionar documentos, comunicarse con el abogado y realizar el seguimiento de sus procesos legales. |
| **CU02** | Citas | Permite al cliente programar citas legales, seleccionar la fecha y modalidad de atención, confirmar la cita y obtener el comprobante correspondiente. |
| **CU03** | Consultas | Permite al usuario registrar una consulta legal, describir su caso, seleccionar el área jurídica correspondiente y recibir la confirmación de la solicitud. |
| **CU04** | Gestionar Perfil | Permite al cliente visualizar y actualizar su información personal, cambiar la contraseña, modificar su fotografía de perfil y mantener sus datos actualizados. |
| **CU05** | Publicación | Permite al abogado crear, editar y publicar contenido informativo, incluyendo texto e imágenes. El administrador puede eliminar publicaciones cuando sea necesario. |
| **CU06** | Servicios Legales | Permite a visitantes y clientes consultar información sobre los servicios legales, acceder a recursos educativos, solicitar citas y visualizar noticias del sistema. |
